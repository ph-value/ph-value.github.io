# [소셜 코딩으로 이끄는 Github 실천 기술] 리뷰

> 부제 - Git과 GitHub를 직접 따라하며 배운다



## 이 책을 읽게 된 계기

실무에서 처음 만난 Github은 꼭 필요하고 간단하게 사용할 법한 명령어만 쓸 줄 알게 되었다.

`git clone  `, `git commit `, `git push`, `git checkout ` ...

그 외 필요한 다른 것들은 Bitbucket을 이용했었다.

두 손에 꼽힐 정도로 적은 수의 명령어들로는 Git/Github을 활용하기에 부족함을 느꼈고 Pull Request를 (!!!) 해보고 싶은 욕망 (!!!)이 들었기 때문에 이 책을 읽기로 했다.

협업에 관련해서도 처음부터 Jira를 이용해 진행했기 때문에 Git/Github을 이용한 커맨드 라인 버전 협업 기술에 대해서도 배우고 싶었다.



인터넷에 많은 튜토리얼들과 실습 위주의 포스팅이 많지 않느냐 하시는 분들도 있겠지만, "한번 써본다" 보다는 "공부한다" 에 초점을 두고 도서 검색을 통해 이 책을 찾게 되었다. 더불어 저자가 담고자 한 연관된 내용을 비롯한 여러가지 이야깃거리들을 만나고 싶었다.
